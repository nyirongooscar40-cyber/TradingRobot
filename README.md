# TradingRobot
 Boom &amp; Crash Spike Detection EA (MetaTrader 5)  This project is a custom MetaTrader 5 Expert Advisor (EA) designed to trade Boom and Crash indices by detecting spike patterns and executing trades based on strict risk management rules.
 # Tools & Technologies
MQL5 (MetaEditor 5) — for algorithmic trading logic and EA deployment

Python — used for backtesting, performance analysis, and data visualization

AI & Machine Learning — applied in spike pattern recognition, risk modeling, and strategy optimization

# Features
Spike detection logic (≥ 5 tick candles)

Automated trade execution (BUY on Crash, SELL on Boom)

Fixed lot size with capped trades per entry

Risk management: stop trading if losses exceed 20% of capital

Minimum capital requirement check ($1)
